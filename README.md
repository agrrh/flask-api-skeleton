# Info

Flask API example.

# Routes

- `GET` `/`

  Index page. Generate human-readable routes index.

  Sample response:

  ```
  GET /uuid/random
   uuid_get_random

  GET /
     index
  ```

- `GET` `/uuid/random`

  Generate random UUID.

  Sample response:

  ```
  26912c10-05cb-492f-847c-e3ebac247c98
  ```
